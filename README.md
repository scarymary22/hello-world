# hello world
* we used brew.sh to install git

## setup git user
* `git config --global user.name "John Doe"`
* `git config --global user.email johndoe@example.com`

## steps
1. `echo "hello world" > README.md`
1. `git status`
1. `git add README.md`
1. `git status`
1. `git commit -m "initial readme"`
1. `git status`
1. `git push`
